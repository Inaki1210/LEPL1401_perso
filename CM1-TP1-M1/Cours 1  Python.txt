INSTRUCTIONS CONDITIONNELLES
avec if et else

if (condition sans la parenthèse) :
	1er bloc (faire quelque chose)
else :
	2ème bloc (faire autre chose)

suite du code

Si il n'y a rien a faire en else, il peut être retiré

APPROCHE FACILITEE POUR CODER UNE CONDITIONNELLE

1. définir tous les cas
2. les ordonner
3. if : cas True
4. elif : plus de deux cas
5. else : cas False
6. La suite du code

INSTRUCTION DE BOUCLE
