README des programmes assistants.py et test.py (V2 sur Inginious)
Créé par Hélène Dodrémont et Iñaki Salomé-Parra, groupe 11.97

PROGRAMME assistant.py
    Le programme fonctionne correctement sur VS code. Aucun message d'erreur ne subsiste.
    Les fonctions liées au fichier 'filemane' sont appelés de tel sorte qu'on puisse utliser des fichier qui n'ont pas d'abord été chargé par file(filename)
    En fait, lorsque que le fichier n'est pas chargé, le programme va d'abord utiliser file(filename) puis la fonction demandée pour éviter un bug lors de l'exécution de la fonction.

PROGRAMME tests.py
    Le programme à l'air de marcher correctement sur VS code. Aucun message d'erreur n'est apparu lors des tests.
    Ce programme utilise les fonction de assistant.py en les important sous le nom de assist.
